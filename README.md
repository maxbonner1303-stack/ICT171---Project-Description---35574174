## Project Overview
This project is a personal subscription tracking service built and hosted on an AWS EC2 Virtual Machine. 
The idea came from a common problem: subscriptions pile up over time and people forget to cancel 
them, leading to unnecessary spending. While enterprise-level tools exist for companies, very few 
solutions exist for personal use.

The site allows users to log their subscriptions, including cost and renewal frequency. It 
calculates total monthly spending and is designed to send a report on the first of each month. 
If the reported total doesn't match what leaves the user's account, it flags that an unaccounted 
subscription may exist.

The goal is to give users a clear, honest picture of where their money is going — and help them 
stay on top of cancellations before costs add up.
